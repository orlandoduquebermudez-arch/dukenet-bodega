¿Cómo configurar?
1. Ve a console.firebase.google.com
2. Crea un proyecto → Realtime Database → Crear base de datos (modo de prueba)
3. Ve a Configuración del proyecto → Aplicaciones web → Registrar app → Copia el firebaseConfig
4. Pega el JSON abajo y guarda.
